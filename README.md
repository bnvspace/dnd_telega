# Telegram бот "Общая повозка"

Бот для группового чата, который ведет одно общее сообщение с инвентарем (например, вещи DnD-партии).  
Он не плодит новые списки, а постоянно редактирует один и тот же пост.

## Что умеет

- Создает одно основное сообщение и хранит его `message_id`.
- Все изменения вносятся через `editMessageText` именно в это сообщение.
- Поддерживает добавление, удаление и очистку.
- При отсутствии списка подсказывает создать его через `/init`.
- Работает корректно при одновременных действиях участников (блокировка по чату).
- Удобный быстрый режим: действия ответом на закрепленное сообщение.

## Команды

- `/start` - в личном чате показывает инструкцию по подключению бота к группе.
- `/init` - создать или восстановить основное сообщение повозки.
- `/add <предмет>` - положить предмет.
- `/del <номер|текст>` - достать предмет по номеру или точному тексту.
- `/clear` - очистить повозку.
- `/show` - принудительно обновить основное сообщение.
- `/help` - показать справку.

Алиасы:

- `/put`, `/loot` = `/add`
- `/take`, `/drop`, `/rm` = `/del`
- `/inv`, `/list` = `/show`

## Быстрый режим (самый удобный в группе)

Сделайте **reply на закрепленное сообщение повозки** и отправьте:

- `+ зелье лечения` - добавить предмет
- `- 2` - удалить второй предмет
- `- веревка 15м` - удалить по точному тексту
- `clear` - очистить повозку

Если сообщение отправлено не ответом на основной список, бот его игнорирует.

## Что видят пользователи в основном сообщении

- Заголовок: `ОБЩАЯ ПОВОЗКА` с custom-emoji.
- Дата обновления.
- Нумерованный список предметов.

Блок с подсказками команд в самом основном сообщении не выводится.

## Запуск

1. Установите Python 3.11+.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте `.env` по примеру `.env.example` или экспортируйте переменные окружения:

- `TELEGRAM_BOT_TOKEN` - токен бота.
- `STATE_DB_PATH` - путь к SQLite БД (по умолчанию `bot_state.sqlite3`).

4. Запустите:

```bash
python bot.py
```

## Рекомендации для группы

- Добавьте бота в группу.
- Дайте право отправлять сообщения.
- Выполните `/init`.
- Закрепите созданное сообщение повозки.
